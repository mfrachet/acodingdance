My toolbelt for safely building features

May 19, 2020

This post won't cover any usage of the tools.

This is a list of tools that I love to use to organize myself or to get confidence about what I ship. Also, I'm mostly building React applications so it might get a bit directed.

Testing automation

  • Jest
  • @testing-library/react
  • Cypress.io
  • cypress-testing-library

Auditing

  • cypress-audit
  • bundlesize
  • code climate

Feature flags

  • LaunchDarkly
  • I would like to try Unleash

Monitoring / alerting / tracking

  • Sentry.io
  • Grafana
  • Google Analytics

UI

  • Emotion
  • Figma
  • Storybook

Organization

  • Franz
  • Rambox

Shared documentation

  • Notion
  • Excalidraw
  • adr-tools

Issue tracker

  • Linear
